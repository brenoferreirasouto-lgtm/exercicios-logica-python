# Cadastro com Validação (Python)

Exercício de lógica em Python que simula um cadastro simples no terminal, com validações básicas de entrada.

## ✅ Validações implementadas
- **Nome:** mínimo de 3 caracteres
- **Idade:** deve estar entre **18 e 120** anos (aceita apenas números)
- **Email:** precisa conter **"@"** (e ter tamanho mínimo)

## 🧠 O que foi praticado
- Laços de repetição (`while True`)
- Validação de entradas do usuário
- Tratamento de erros com `try/except`
- Manipulação básica de strings (`strip`, `len`, verificação com `in`)
- Saída formatada com `f-string`

## ▶️ Como executar
1. Tenha o Python instalado (versão 3+)
2. No terminal, navegue até a pasta do arquivo e execute:

```bash
python cadastro.py
