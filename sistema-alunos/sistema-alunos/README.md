# Sistema de Cadastro de Alunos 🎓

Sistema de menu interativo em **Python** para cadastro e gerenciamento de alunos via terminal.

## ✅ Funcionalidades
- Cadastrar aluno com nome, idade e 3 notas
- Calcular média automaticamente
- Classificar como **APROVADO** (média ≥ 7.0) ou **REPROVADO**
- Listar todos os alunos com situação
- Buscar aluno por nome (busca parcial)

## 🛡️ Validações implementadas
- Opção inválida no menu tratada com mensagem de erro
- Idade inválida (letra) tratada com `try/except`
- Nota inválida (letra ou fora de 0–10) tratada com `try/except`

## 🛠️ Tecnologias
- Python 3
- Estruturas: `list`, `dict`, `while`, `for`, `try/except`

## ▶️ Como executar
```bash
python sistema-alunos.py
